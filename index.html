Using Functions

The form validation library includes a range of functions that can help you validate different types of data. Here's how to use them:

  1. Include the library in your code: <script src="https://iva-stasia.github.io/validation-library/validationLibrary.js"></script>
  2. Call the validation function for the data you want to validate: const email = document.querySelector('#email').value;
    const validatedEmail = validateEmail(email);

As a result of validation, you get an object that contains the isValid key with a boolean value, and the message key that contains an array with messages about what exactly is wrong with the value.

Haw can you use functions?

Example:

<form>
  <label for="email">Email:</label>
  <input type="email" id="email" name="email" />
  <div class="invalid-feedback">The email is required.</div>
  <button type="submit">Submit</button>
</form>

<script>
  const email = document.querySelector('input[type=email]');

  email.addEventListener('change', () => {
    const validatedEmail = validateEmail(email.value);
    const messageField = document.querySelector('.invalid-feedback');

    if (!validatedEmail.isValid) {
      email.classList.remove('is-valid');
      email.classList.add('is-invalid');
      messageField.innerHTML = validatedEmail.message[0];
    } else {
      email.classList.remove('is-invalid');
      email.classList.add('is-valid');
    }
  });
</script>

What can you validate?

  • Numbers
    const validatedNumber = validateNumber(number);
    const validatedNumber = validateFloatingPointNumber(number);
  • Text
    const validatedText = validateText(text);
  • Name, username, password
    const validatedName = validateName(name);
    const validatedUsername = validateUsername(username);
    const validatedPassword = validatePassword(password);
    const isPasswordMatch = validatePasswordMatch(password, repeatedPassword);
  • Email
    const validatedEmail = validateEmail(email);
  • Date
    const validatedDate = validateDateMonthDayYear(date);
    const validatedDate = validateDateDayMonthYear(date);
  • Phone number
    const validatedPhone = validatePhoneNumber(phoneNumber);
  • Credit card
    const validatedCardNumber = validateCardNumber(cardNumber);
    const validatedCardCVV = validateCardCVV(cardCVV);
